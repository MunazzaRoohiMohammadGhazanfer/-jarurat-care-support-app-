# Jarurat Care – Mini Healthcare Support Web App

*Note: This is a concept-level prototype focused on clarity and effort, as per assignment guidelines.*

**Tech Stack:** React.js, Node.js, Express.js, CSS3

**Live Demo:** `Add your Vercel link here after deployment`

### AI / Automation Ideas

1. **FAQ Chatbot:** A rule-based chatbot that answers common queries on visiting hours, registration process, and emergency contact. Reduces repetitive manual work for NGO staff.

2. **Auto-response & Data Summary:** Sends instant email confirmation after form submit. Auto-generates a live count of patient vs volunteer requests for admins to track demand.

### NGO Use-Case

This app helps Jarurat Care digitize patient/volunteer intake and handle FAQs 24/7. Automating initial queries and data collection lets staff focus on critical support instead of repetitive form processing.

### How to Run Locally
1. Clone the repo
2. Run `npm install` 
3. Run `npm start`
